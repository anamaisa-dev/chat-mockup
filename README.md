# Chat Mockup

Projeto desenvolvido em 2021 como um experimento pessoal para testar minha capacidade de simular o fluxo básico de um aplicativo de mensagens.

Ao acessar o site, o usuário encontra uma interface simplificada de chat: é possível enviar qualquer mensagem, que aparece com o horário do envio. Após alguns segundos, o sistema retorna automaticamente uma mensagem de resposta — um texto padrão (“lorem ipsum”) definido no código — marcada com um horário dois minutos à frente.

O objetivo era experimentar conceitos como manipulação de estados em aplicações front end, exibição dinâmica de mensagens na tela e criação de respostas automáticas simuladas, inspirando-se em aplicativos populares de mensagens.

Este exercício prático me permitiu explorar técnicas de UI/UX, trabalhar a lógica de temporização de respostas e controlar fluxos assíncronos de interação, todos importantes para o desenvolvimento web moderno.

Veja o resultado: https://anamaisa-dev.github.io/chat-mockup/
